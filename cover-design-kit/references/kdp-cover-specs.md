# KDP Cover Specs (April 2026)

## Ebook

- Minimum 2560×1600 px
- Aspect ratio 1.6:1
- JPG or TIFF
- RGB, sRGB color profile
- Under 50MB

## Paperback (full wrap)

- Single PDF wrap file
- Trim: **5.25×8** for cozy fantasy (5.5×8.5 acceptable, 6×9 avoid — wrong genre feel)
- Cream paper (fiction standard)
- Matte finish (warmer than gloss)
- Bleed 0.125" if art extends to edge (for text-only interior, no bleed needed)
- Spine width formula:
  - **Cream paper:** `pages × 0.0025"`
  - White paper: `pages × 0.002252"`
  - Example: 320-page book on cream = 0.8" spine
- Use KDP Cover Template Generator — input trim + page count, get exact PDF template

## Hardcover (when applicable)

- Case laminate construction
- Trim 6×9 or 5.5×8.5
- Separate spine calc (thicker, case-laminate specific)
- KDP template generator handles

## File naming convention

```
[Series]_Book[N]_Ebook_v[x].jpg
[Series]_Book[N]_Paperback_5.25x8_[pages]pp_v[x].pdf
[Series]_Book[N]_Hardcover_6x9_[pages]pp_v[x].pdf
```

## A+ Content assets

- Title banner: 970×600 px
- Character/setting images: sized per Amazon module specs
- Comp-authors banner, author bio banner — per module specs

## Social reveal pack (deliver with cover)

- Hi-res cover
- 3D-on-device mockup (Book Brush or equivalent)
- Square social crop
- Landscape social crop
- Story-sized (9:16) crop
- Optional: animated reveal if budget allows
