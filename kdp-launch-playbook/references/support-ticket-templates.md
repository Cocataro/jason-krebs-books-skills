## Pre-order open-date change

```
Subject: Pre-order release date change for ASIN [ASIN]

Hello KDP team,

I need to adjust the release date for my pre-order: [Title] (ASIN [ASIN]).

Current release date: [date]
Requested new release date: [date]

The change is due to [reason — editorial / production]. Please confirm this adjustment is processed at least 3 days before the current release date to avoid delivery issues.

Thank you.

[Pen name]
```

**⚠ If less than 7 days from current release date, this is an urgent escalation. Missing pre-order delivery = 1-year ban.**
