# Forbidden Keywords (Amazon KDP TOS)

Verify the full current list at every 90-day refresh — Amazon updates periodically.

## Always forbidden

- **Other author names** — any other author, living or dead, by any spelling. TOS violation, delist risk.
- **Book titles** — any other book, yours or anyone else's, verbatim.
- **Trademarked brands and IP** — Harry Potter, Tolkien, Sanderson, Game of Thrones, Marvel, Disney, etc. Immediate delist risk.
- **"free"** — not a valid keyword.
- **"bestseller"** — not a valid keyword.
- **"new"** — not a valid keyword.
- **"award-winning"** — unless true AND substantiated with external verification.
- **Time-bound terms** for evergreen fiction: "Christmas 2026", "Valentine's Day 2025" — if the book isn't seasonal, these rot fast.
- **Explicit content tags** for non-explicit content — misrepresentation, refund risk.
- **Political / religious inflammatory terms** unless directly relevant to the book's content.

## Gray area (avoid by default)

- Author-genre comps: "Travis Baldree style" — describes flavor, doesn't quote. Borderline. Prefer "cozy fantasy coffee shop" framing instead.
- Book-title echoes: "Legends and Lattes readalike" — describes genre. Generally safe but risk of flagging. Prefer generic "cozy fantasy inn" phrasing when in doubt.

## Safe reframings

If tempted to use a forbidden term, reframe to describe the vibe:

| Tempted | Safer |
|---|---|
| "fans of Travis Baldree" | "cozy fantasy coffee shop" / "found family cozy fantasy" |
| "Legends and Lattes readalike" | "cozy fantasy shop-opening story" |
| "Harry Potter style magic school" | "whimsical magic academy fantasy" |
| "Tolkien-esque" | "high fantasy worldbuilding" / "epic fantasy adventure" |
| "Christmas 2026 read" | "winter holiday cozy fantasy" |

## If a keyword gets flagged

- Amazon sends a warning email. Sometimes the book is quietly delisted.
- Immediate remediation: remove the keyword from all affected books; respond to Amazon with the fix.
- Document the incident for future avoidance.
- Escalate to CEO Jason immediately — account warnings compound.
