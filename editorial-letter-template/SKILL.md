# Editorial Letter Template

Target length: **7,000–9,000 words** for a 90k manuscript. Under 3k = too shallow; over 15k = unfocused; ghostwriters stop reading past page 20.

Read `references/scene-grid-template.md` before writing — the scene grid is built before any part of the letter.

## The 12 Sections (rigid order)

### 1. Opening / love letter (300–500 words) — **NOT OPTIONAL**
Specific strengths. What's working. Named scenes. Editors who skip this lose the writer.

### 2. Big-picture diagnosis (500–800 words)
Name the ONE core structural issue. Rank if multiple.

### 3. Structural / plot analysis (1,500–2,500 words)
Beat-by-beat against the Heartland Fantasy rubric + Save the Cat. Where acts break, where pacing sags, where setups lack payoffs.

### 4. Character arcs (1,000–1,500 words)
Protagonist internal change, supporting cast agency, antagonist (in cozy: circumstance or soft-human, never a villain), found-family dynamics.

### 5. POV & voice (300–600 words)
Consistency, narrative distance, warmth-of-narrator maintained.

### 6. Pacing & scene economy (400–800 words)
Scenes to cut, compress, merge. Cozy tolerates more hangout than thriller, but hangout must still do work.

### 7. Dialogue (200–500 words)
Distinctiveness per character, subtext, exposition dumps.

### 8. Worldbuilding & setting (300–600 words)
Is the inn a character? Do objects carry meaning?

### 9. Genre fit & marketability (400–700 words)
Comp check (still in Baldree / Yarros-adjacent lane?). Tropes hit. Cover-copy-able hooks.

### 10. Humanizing-fiction check (200–400 words)
Rhythmic variance, idiosyncratic observation, unearned-clean-beat flags. **Structural AI-tells** to scan for: beats landing too precisely on percentages; sentence-length variance too low; observations too general; scene-end reflection sentences summarizing theme.

### 11. Revision priority list (300–500 words)
Numbered. **P1 = must-fix (cap at 5), P2 = should-fix, P3 = optional polish.** Scope so revision is 2–3 weeks max.

### 12. Closing (150–300 words)
Reaffirm the book's promise. Name 1–2 scenes to preserve untouched.

## Exec summary to EIC (separate one-pager, delivered with the letter)

```
MS: [Book N — Title] | Word count: [n]
Verdict: [ship-ready | revision-required | fatal]
Top 3 issues: [numbered, with severity P1/P2/P3]
Estimated revision time: [weeks]
Cross-book arc risks: [any]
Humanizing-fiction structural check: [pass | flagged]
```

## Process

1. Cold read, no notes, one sitting if possible (6–8h for 90k).
2. 200-word gut-reaction memo (private).
3. Sleep on it.
4. Scene grid / structural read (see `references/scene-grid-template.md`).
5. Letter draft, 12 sections in rigid order.
6. Inline margin notes ONLY on scenes the letter references — never copy-level.
7. **Hand to EIC for QA.** Do NOT send directly to Series Writer.
8. On EIC approval: release to Series Writer + kickoff brief. Clarification window: 1 week.

## Hard rules

- Love-letter section is non-negotiable.
- P1 cap at 5. More than 5 must-fixes = rewrite, not revision.
- No line-edits in margin notes.
- EIC gates every letter. No exceptions.
- Cite chapter + paragraph + line when pointing at issues.
- Humanizing-fiction check always present (§10), even if clean.
