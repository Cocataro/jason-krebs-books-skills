# Scene Grid Template

One grid per book. Retained across the series. Populated during Read 2 of the dev edit.

## Columns

| # | POV | Location | Purpose (1 line) | Stakes (personal/relational/local) | Emotional beat | Word count | Genre beat served | Flags for letter |

## Rules

- One row per scene (not per chapter; chapters may contain multiple scenes).
- **Purpose** must fit in one sentence. If you can't, the scene probably needs cutting.
- **Stakes** — rate against the cozy ceiling. Any row marked "regional / national / world / cosmic" is a red flag.
- **Genre beat served** — which of the 7 cozy fantasy rubric beats does this scene serve? Or mark "transitional" / "character" / "worldbuilding." Any scene that serves no beat and isn't clearly transitional/character/worldbuilding is a cut candidate.
- **Flags** — one-line note of anything letter-worthy (pacing, voice drift, canon concern, humanizing-fiction issue).

## How to use

1. **Pattern-hunt** the completed grid before writing the letter.
2. **Warmth floor check:** are there any stretches of >2 chapters without a warmth-positive scene? Flag.
3. **Stakes ceiling check:** any "regional/national/world" rows? Flag.
4. **Beat coverage:** are all 7 rubric beats served? Any missing → §3 of the letter (Structural/plot analysis).
5. **Scene economy:** which purpose-less rows exist? → §6 Pacing.
6. **Series laddering:** do any scenes plant a seed that pays off in Book N+1 or Book N+2? Note them — don't cut them even if they look thin in isolation.
