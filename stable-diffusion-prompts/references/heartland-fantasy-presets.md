## Rules for use

1. These are STARTING points. Edit specifics per book / scene / moodboard.
2. Never ship these outputs as final covers (HARD GUARDRAIL in `SKILL.md`).
3. For illustrator reference, generate 8–12 variants, select 2–3 that land the mood, send to illustrator as inspiration (not imitation target).
4. For marketing use: see `marketing-imagery-kit` skill for acceptable-use rules.
5. Log final prompt + seed + model per saved image for reproducibility.
