# Style-Sheet Structure

Two-file pair in studio Drive + mirrored to vault.

## Google Sheet: Character/Term Registry

Columns:

| Entry | Type | Canonical Spelling | First Appearance (book:ch) | Capitalization | Italicization | Plural Form | Notes |
|---|---|---|---|---|---|---|---|

Types: `character`, `place`, `term`, `phrase`, `dialect_marker`, `recipe`, `in_world_object`

## Google Doc: Prose Rules + Narrative Descriptions

### Section 1: Characters (Deep)

For each major character:
- Full name + preferred spelling
- Aliases / nicknames / titles
- Physical: eye color, hair color, height, build, scars, distinguishing marks, age-progression per book
- Verbal: speech register, filler words, catchphrases, regional markers, contraction usage, profanity register
- Cognitive: interior-thought treatment (italic/roman), thought patterns, verbal tics (restart sentences, fragment usage)
- Wardrobe defaults
- Object associations (Briar's iron ring, Gorrick's scarred hands, etc.)
- Age-per-book chronology

### Section 2: Invented Terms

For each term:
- Spelling (canonical — one forever)
- Capitalization rule
- Italicization rule (first-use italic → roman; or always italic; or always roman)
- Plural form
- First appearance (book:ch)
- Pronunciation note (for audiobook eventual)
- Etymology / in-world origin (optional)

### Section 3: Place Names

Same treatment as invented terms. Plus:
- Geography / location within series world
- Pronunciation
- Notable in-world features

### Section 4: Timeline

- Series calendar (months, seasons, named festivals)
- Cross-book chronology (Book 1 starts in [season]; Book 2 is [time elapsed] later)
- Character ages per book
- Major historical events (war, founding dates, etc.)

### Section 5: Dialect Rules Per Character

- Contractions allowed / forbidden
- Regional markers (specific to POV or speakers from certain regions)
- Formality level (formal / colloquial / register shifts)
- In-world idioms associated with this character / region

### Section 6: Formatting Conventions

- Chapter header style
- Scene break glyph (❦, ⁂, or custom)
- Inner thought treatment (italic / roman)
- Telepathy / magical communication treatment
- Written-in-story material (letters, signs): quoted / italicized / block
- Em-dash vs. en-dash usage
- Ellipsis spacing (per CMOS or per humanizing-fiction exception)
- Italics for emphasis vs. made-up words vs. foreign phrases

### Section 7: CMOS Exceptions (House Rules)

Where the studio deliberately breaks CMOS for humanizing-fiction reasons:
- Fragment acceptance (specific patterns)
- Comma splice acceptance (in specific emotional contexts)
- Run-on sentence acceptance (anxiety / breathless beats)
- Serial comma (house: YES, per CMOS)
- Any other documented deviations

### Section 8: Recurring Phrases / Catchphrases

- Character catchphrases
- In-world idioms
- Verbal tics that recur across books (Briar's "honest" as filler; her "I don't run. I don't hide. I don't break." prayer)

### Section 9: Series Bible Link

- Pointer to the broader worldbuilding doc
- Cross-reference to canonical character bible

### Section 10: Style Sheet Delta Log

- What changed this book (new characters, new terms, new rules)
- Date + editor + book reference
- For audit trail
