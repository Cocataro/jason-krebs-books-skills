# Query Templates

Use these as starting points. All queries prefix with `[Q]` in gdoc margin comments.

## Fragment / intent query

```
[Q] Fragment — intentional? (see similar pattern on pp. X, Y — propose codifying in style sheet §7 if house)
```

## Run-on / rhythm query

```
[Q] Run-on during [emotional beat] — preserve for breathlessness, or split? Preserve preferred unless it loses readability.
```

## Invented term spelling

```
[Q] "[Term]" — style sheet has "[Canonical]". Which is current canon?
```

## Modern idiom flag

```
[Q] Modern idiom "[word/phrase]" — per style guide §[X], preserve if intentional. Confirm.
```

## POV drift

```
[Q] POV drift — paragraph shifts to [other character]'s interior. Reframe through [POV character] or intentional close-omniscient?
```

## CMOS departure

```
[Q] CMOS 6.X prefers [Y]. Style guide allows [departure]. Confirm preserve?
```

## Continuity flag

```
[Q] Continuity — Book 1 chapter 3 has [X]. This says [Y]. Which is canonical? Style-sheet entry needed.
```

## Potentially-inconsistent dialect

```
[Q] Character [X] uses [form A] here but [form B] earlier (p. Z). Dialect inconsistency or intentional shift (emotional state, addressing different character)?
```

## Timeline question

```
[Q] Timeline — this scene says "[N] days ago" but Book [N] chapter [X] established [event] at [time]. Which is canonical?
```

## Character-detail inconsistency

```
[Q] Character detail — [X] said [A] in Book [N] ch [X]; this says [B]. Which is canonical?
```

## Author's note on unresolved

When a query can't be resolved at copyedit, note in editor's note at handoff:

```
Open queries for EIC ruling:
1. [Query summary] — location [p. X] — impact: [scope]
2. [Query summary] — location [p. X] — impact: [scope]
```

## When to NOT query

- Clear homophone errors (their/there/they're, its/it's) — fix
- Missing-word errors — fix
- Obviously reversed punctuation (commas outside closing quotes in US style) — fix
- Invented-term misspelling if the style-sheet entry is clear — fix + log
- Timeline contradiction where the correct value is clear from prior canon — fix + note

Over-querying signals a timid editor. The target is 10–30 queries per 90k. Not "query everything that might be wrong" — query specifically when intent is ambiguous.
