# Style-Sheet Template

The style sheet is the editorial memory of a series. It grows with every chapter across every book. By Book 6 expect 15–25 pages.

See `references/style-sheet-structure.md`, `references/preserve-vs-query-rule.md`, `references/cmos-18-anchors.md`, `references/query-templates.md`.

## Core Rule: Humanizing-Fiction > CMOS

The Series Style Guide mandates deliberate imperfections — fragments, run-ons, regional idiom, uneven paragraph length, authorial tics. A default-trained editor will "fix" these out. **That is the #1 failure mode for this role.**

**Preserve by default when:**
- Pattern recurs (a fragment once = probably error; a fragment every 3–5 paragraphs = voice)
- Contextual function (run-on during anxiety = earning its keep; run-on in exposition = probably not)
- Style-sheet precedent (already codified → preserve)
- Author signal (tics declared in style guide are invariant)

**Edit only when:**
- Produces genuine reader confusion
- Breaks continuity
- Violates an explicit style-sheet rule

**When uncertain: QUERY, don't CORRECT.** Silent-correcting is a terminating offense.

## Style Sheet Structure

**Two-file pair:**
- **Google Sheet** — fast lookups (character/term registry)
- **Google Doc** — prose rules + narrative descriptions

See `references/style-sheet-structure.md` for full schema.

## Growth Discipline

- Every new character → entry on first appearance
- Every invented term → entry on first appearance  
- Every dialect rule decision → entry in doc
- Every CMOS exception → entry in doc
- Mirror to vault markdown weekly

## Query Targets

- **Query rate per 90k novel: 10–30.** Too few = over-editing. Too many = timid.
- **Never query a clear error** — just fix.
- **Always query a potentially-intentional imperfection** — never silent-fix.
- **Query templates:** see `references/query-templates.md` for reusable phrasings.

## Handoff to EIC

At pass-completion, deliver:
- Tracked-changes manuscript (accepted except open queries)
- Updated style sheet (doc + sheet versions)
- Editor's note: unresolved judgment calls for EIC ruling
- Style-sheet delta this book (what was added)
- Voice-preservation confidence rating
- Open queries count

## KPIs

- Line edit throughput: 1,000–1,500 wph
- Copy edit throughput: 5–10 pages/hr (1,250–2,500 wph)
- Combined pass on 90k manuscript: 3–4 weeks calendar, 70–90 hours work
- Post-edit error density: <1 error per 2,000 words reaching proof
- Style-sheet completeness: 100% of named entities captured
- Cross-book consistency: >95% (spot-check 50 invented terms)
- Voice-preservation: >90% (EIC audit)
- Query rate: 10–30 per 90k

## Hard Rules

- No silent corrections on intent-possible imperfections. Terminating offense.
- Style sheet grows every chapter, not every book.
- CMOS 18 is the fallback, not the master. Humanizing-fiction directive wins.
- EIC signs off before handoff to Proofreader or Formatter.
- Invented-term spellings are invariant once codified. Zero drift tolerance.
